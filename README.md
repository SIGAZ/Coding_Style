# 변수/함수 작성 스타일

## 변수/함수 표기법
파스칼 표기법 + 스네이크 표기법


스네이크 표기법으로 공백을 언더바로 표시하고 가독성을 쉽게하기위해 단어 첫 문자는 대문자로 작성


```
ex) Pascal_Snake
```
